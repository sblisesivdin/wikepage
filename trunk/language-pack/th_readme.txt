Wikepage 2007.2 Opus 13 "Landauer-Bttiker" Wiki/Blog Hybrid Engine
__________________________________________________________________________

วิกีเพจไทย 2007.2 รุ่น Opus 13 Wiki/Blog Hybrid Engine แปล พัฒนาและปรับแก้ เพิ่มเติมโดย ผศ.ประชิด ทิณบุตร 

แนะนำระบบ : Introduction
Wikepage คือ โปรแกรมวิกีมาตฐาน ที่ง่ายต่อการใช้งาน เป็นลกผสมระหว่างวิกีและบล๊อกที่มีขนาดเล็ก ที่พัฒนาต่อจากTipiwiki2 (http://tipiwiki.sourceforge.net/). Wikepage มีระบบความปลอดภัยบางอย่างที่ทาง
Tipiwiki ยังไม่มี, เช่นการปกป้องโครงสร้างของรหัสผ่าน ด้วยการอุดช่องโหว่ของระบบความปลอดภัย ที่สำคัญไว้แล้ว, เพิ่มระบบคำสั่งสนับสนุนการนำเข้า และการส่งออกไฟล์ RSS feed สนับสนุนการสร้างตาราง( table support), การส่งไฟล์ขึ้นระบบ(upload support),การใช้งานหลายภาษา(multilanguage) และการสนับสนุนการทำงานร่วมหลายพื้นที่(multisite support)..เป็นต้น...


Wikepage มีการทำงานอยู่ 2 รูปแบบภาวะ(Mode)คือ:

-รูปแบบภาวะวิกี (Wiki Mode): หมายถึงหน้าเอกสารหรือบทความในบล๊อก(Pages and Blog)ใดที่สร้างไว้และสามารถเข้าแก้ไขได้โดยอิสระ หรือไม่ต้องเข้าระบบด้วยรหัสผ่านแต่อย่างใด หรืออีกนัยหนึ่งก็หมายความว่า ใครๆก็สามารถเข้ามาเพิ่มเติมหรือแก้ไขเอกสารที่มีได้โดยอิสระเสรี.
-รูปแบบภาวะส่วนบุคคล(Personal Mode): หมายถึงหน้าเอกสารหรือบทความในบล๊อก(Pages and Blog)ใด ที่มีผู้สร้างไว้ สามารถกำหนดการปกป้องสิิทธิ์การเข้าแก้ไขเนื้อหาที่ตนเองสร้างเอาไว้ได้ หรือโดยที่ต้องมีรหัสผ่านเข้าก่อนจึงจะสามารถทำการแก้ไขได้

*INSTALL: การติดตั้งระบบ
-เปิดไฟล์ index.php ด้วยโปรแกรม notepad หรือโปรแกรม editor อื่นๆ เพื่อเปลี่ยนข้อมูลเกี่ยวกับเว็ปไซท์ของคุณ ภาษาและการตั้งค่าอื่นๆของเว็ปไซท์คุณ.(เปลี่ยนภายหลังก็ได้)
-ส่งไฟล์ทั้งหมดขึ้นไปเก็บที่แฟ้มเก็บในเครื่องแม่ข่ายบริการที่คุณใช้ด้วยโปรแกรมประเภท FTP client.
-เปลี่ยนโหมดการบันทึกไฟล์ ให้สามารถบันทึกเข้าได้ (Chmod 777) (อย่างน้อยต้องเป็น 755 แต่บางครั้งอาจจะใช้ไม่ได้) ของไฟล์ที่อยู่ในแฟ้ม data และโฟลเดอร์ย่อยทั้งหมด. และไฟล์ passwd.php
-เครื่องแม่ข่ายบริการของคุณต้องติดตั้งโปรแกรม PHP และกำหนดค่าคำสั่ง safe_mode.ในไฟล์ php.ini ของเครื่องแม่ข่าย(Server) บรรทัดคำสั่ง safe_mode เป็น safe_mode=off หาก เป็น on โปรแกรมจะไม่ทำงานร่วมได้.
-หากคุณต้องการเพิ่มภาษาอื่นๆให้ดาน์โหลดได้จากที่ http://www.wikepage.org/  หรือที่ http://www.prachid.com สำหรับภาษาไทย
 และให้คลายไฟล์ออกและส่งไฟล์ขึ้นเก็บในโฟลเดอร์ที่กำหนดเก็บไฟล์ภาษาไว้ที่เดียวกับภาษาอื่นๆ. หลังจากนั้นให้เปลี่ยนภาษาเริ่มต้น ได้โดยการเปลี่ยนค่าตัวแปรในไฟล์ index.php. และถ้าหากต้องการให้ใช้ได้หลากหลายภาษา คุณสามารถแก้ไข หรือเพิ่มเติมบรรทัดคำสั่งเชื่อมโยงใช้สลับภาษาได้ง่ายๆ ภายในเว็ปไซท์ของคุณ โดยแก้ [index.php?lng=en|English]
 [index.php?lng=tr|Turkce] ในไฟล์เมนูเลือกใช้ภาษา.
-การเข้าเว็ปไซท์ของคุณ. ครั้งแรกจะเข้าสู่หน้าการจัดการ(Admin page)โดยอัตโนมัติ. ซึ่งคุณจะต้องตั้งรหัสผ่านในฐานะผู้บริหารจัดการระบบก่อนตามที่ต้องการตั้ง .
-เว็ปไซท์ของคุณก็จะเข้าสู่ภาวะการใช้งานแบบ Wiki mode!เป็นภาวะแรก
-คุณสามารถเปลี่ยนภาวะการใช้งานในหน้า "การเข้าระบบจัดการ:บล๊อก" จากเมนูแถบมุมขวาบน โดยแจ้งรหัสผ่านของคุณ.
-ในภาวะของ wiki mode คุณต้องใส่รหัสผ่าน "ทุกครั้ง" ที่ทำการเปลี่ยนแปลงในหน้า.
-หากคุณต้องการที่จะใช้บล๊อก/สร้างบล๊อกใหม่ สามารถสร้างจากหน้า"การเข้าระบบจัดการ:บล๊อก" คลิกที่คำสั่ง " New Blog Entry " ที่อยู่ส่วนล่างซ้ายของหน้า [index.php?Blog_Entry|Submit new link] type link.
- คุณสามารถเข้าอ่านบล๊อกได้โดยทาง"<blog_view>" ในทุกหน้าเอกสารบล๊อกของคุณ.

คุณสามารถเข้าอ่านเอกสารเพิ่มเติมได้จากเว็ปไซท์ของคุณเอง และหรือจากในรายการที่เชื่อมโยงไว้.

*IN ADDITION:คำแนะนำเพิ่มเติม
-หากคุณเปลี่ยนโหมดค่าอนุญาตการบันทึกไฟล์ให้กับโฟลเดอร์(chmod)ได้ทุกโฟลเดอร์ wikepage สามารถทำงานได้ไม่น้อยกว่า 90% .
-หากการสร้างไฟล์ใหม่(File create) การส่งไฟล์ขึ้น(upload)และการลบ(delete)ไม่ทำงาน อาจเป็นเพราะเครื่องแม่ข่ายบริการ(Server)ที่คุณเช่า/ใช้งาน ได้ตั้งค่าคำสั่งของไฟล์ php.ini ในบรรทัดคำสั่ง safe_mode=on ไว้ ควรแก้ไข ให้เป็น=off ระบบจึงจะสามารถสนับสนุนการทำงานได้
